Email Genius Chrome Extension

EmailGenius is an AI powered Chrome extension that helps users reply to email, check grammar and translate text. I was responsible for designing both the branding graphics and user interface.

My Process

  1. Brand Design
  2. User Flow
  3. Thumbnails and Wireframes
  4. Prototype

Brand Design

I designed the logo and a style guide to create a cohesive look for the extension.

Logo Design

Style Guide

User Flow

I created a user flow to brainstorm all the required screens. This was also presented to the client to ensure we were on the same page.

Wireframes

After establishing the user flows and required screens, I moved onto designing layouts for each screen. I created thumbnails sketches to quickly test different layouts and then translated those sketches into high fidelity wireframes.

Prototype

Since the client is a developer, I created a prototype to illustrate the possible interactions the extension could have.
