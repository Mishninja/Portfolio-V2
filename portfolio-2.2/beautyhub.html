Beauty Hub Landing Page Mock Up

A landing page for a clean beauty brand.

Overview

View the mobile prototype on Figma

  • An adaptable eCommerce website layout with a cohesive color scheme.
  • The banner features a clear call to action to draws consumers in.
  • The navigation layout allows for more links to be added on for future website expansion.
  • The mobile view was designed on a 3 column grid and desktop view was designed on a 12 column grid.
  • Hover effects were created using variants in Figma.
  • All product mock-up images were photoshopped stock images

Navigation in mobile view. The menu slides in and out from the right.

Beauty Hub banner with clear call to action.

Beauty Hub Featured products section (turns into a carousel in mobile view) and values section.

Beauty Hub announcement section, mailing list sign up section and footer.

Featured Products section in mobile view is a carousel. The sides of the next slides can be seen to let users know that it is scrollable.

Takeaways

  • How to use variants to prototype interactions
  • Prototyping navigation interactions.
  • How to create effective landing pages with room for change.
  • Using different components for different viewport widths.
  • Keeping in mind what how the design would be implemented with code.
