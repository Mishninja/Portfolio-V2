Approach

This webpage was built using a mobile first approach. I first decided the layouts for a mobile view and then scaled everything up. I used media queries for each major breakpoint (320px, 567px, 768px, 992px, 1200px, and 1440px). At each break point I set out to use a layout that made the most sense.

One notable feature is the navigation bar and burger menu. The burger menu was implemented for a mobile view while the navigation bar was implemented for a desktop view. Explore the site.

Features

Accordion

This section was meant to emulate a common FAQ layout. The accordion was implemented using JavaScript and CSS to manipulate divs. An accordion makes associating question and answers simple and clean.

Isotope/Masonry

Isotope allows you to sort through different photos using the categories filters at the top of the section. The responsive layout is built into isotope so the layout of images responds to the width of the screen.

Gridder

Gridder was used in this section to display facts about endangered species in a tidy interface. The images have hover effects on them to let users know that he element is clickable. All color and styling was customized to fit the theme of this web page. The responsive layout of this section was accomplished using media queries.

Responsive Burger Menu

The responsive burger menu was created so that in a mobile view, the navigation bar is hidden and becomes a drop down menu. A combination of media queries and JavaScript was used to create this navigation. In desktop view, the navigation bar appears and sticks to the top of the page. While scrolling down, the bar will disappear. It will reappear when scrolling back up.
